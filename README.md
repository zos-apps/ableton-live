## About

Professional digital audio workstation for music production, live performance, and sound design.

## Features

- Session View
- Arrangement View
- MIDI editing
- Audio effects
- Instrument racks

## Installation

### From App Store
Open the **App Store** in zOS and search for "Ableton Live".

### Via npm
```bash
npm install @zos-apps/ableton-live
```

## Usage

- Create tracks with + button
- Drag samples to clips
- Use transport controls to play

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Stop |
| `Tab` | Switch views |
| `Cmd+S` | Save |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
